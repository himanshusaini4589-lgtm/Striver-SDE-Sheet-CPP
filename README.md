## How to Compile & Run

```bash

```

## Naming Convention

- Folder: `NN-TopicName` (already numbered above)
- File: `lowercase-hyphenated-problem-name.cpp` (e.g. `next-permutation.cpp`)
- No `sol1.cpp`, `temp.cpp`, `final.cpp` — keep it clean, recruiters browse this.
